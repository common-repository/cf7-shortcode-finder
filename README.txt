== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "CF7 Shortcode Finder"
3. **Activate** CF7 Shortcode Finder from your Plugins page
4. **Click** under the contact tab click on "Shortcode Pages" and see the shortcode used on which pages!
